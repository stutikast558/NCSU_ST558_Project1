The Mighty 4

Introduction

Since 2018, most investors and traders around the world are closely watching which Tech Company will hit the Trillion dollar mark. Eventually, Apple made a touchdown on the finish line when their stock price hit an all-time high of $208.74 per share on intra-day trading on the 2nd of August, 2018 (NASDAQ.)

Apple had the glory, cruising at the top alone, but they didn’t enjoy this lonely stay at the top for long. Amazon followed in the rally immediately after, hitting the trillion dollar mark just one month after Apple had done the unbelievable. Microsoft became the third company to complete the Trillion race. The eyes of every market watcher were on the four tech giants since the race to a trillion has started. Alphabet (Google’s parent company) hasn’t made it there yet, but it already worth more than $800 billion and, while well short of Apple, it is now jumping into that batch of companies that are on their way to being a $1 trillion company. For details, read the article on the Race to One Trillion.

Data

To analyze the historical stock market performance of the Mighty 4 , I obtained quarterly Market Sentiment and Open/Close values of the four stocks from 1994 to 2017 in JSON format from public datasets source kaggle

Process and Packages

The four JSON files for four companies have been read into R using rjson package. Though there are two other packages, jsonlite and RJSONIO, I chose the standard and easy to use functions from rjason pacjage because RJSONIO has reported performance issues when handling large JSON files and jsonline is specially useful when using APIs where as my project has downloaded JSON files from Kaggle.

Using the function fromJSON, I read four JSON files for four stocks; the obtained list objects have been converted into data frame and all four data frames have been merged. Additional columns such as company and Percentage change for Open/Closing value of the stock have been calculated.

library(rjson)

ls_msft <- fromJSON(file="MSFT_info.json")

# Merging the dataframe as one combined frame
df_merge1 <- rbind(df_apple,df_amzn)
df_merge2 <- rbind(df_goog,df_msft)

# Set Column Names
names(df_all4)[1] <- paste("Date")
names(df_all4)[2] <- paste("Sentiment")
names(df_all4)[3] <- paste("Open")
names(df_all4)[4] <- paste("Close")
# Update the Column attributes accurately
df_all4$Date  <- as.character(df_all4$Date)
df_all4$Sentiment  <- round(as.numeric(as.character(df_all4$Sentiment)),4)
df_all4$Open  <- round(as.numeric(as.character(df_all4$Open)),2)
df_all4$Close  <- round(as.numeric(as.character(df_all4$Close)),2)
df_all4$Company <- as.factor(df_all4$Company)
#Calculate Open/Close Change Percentage
df_all4_final <- dplyr::mutate(df_all4,Change_Percentage=round((((df_all4$Open-df_all4$Close)/df_all4$Open)*100),2))
# Function to calucalte Summary stats 
getIQR <- function(input1)
{
  subset1 <- filter(df_all4_final, Company == input1) %>% select (Sentiment,Open,Close,Change_Percentage)
  p <- as.numeric(gsub('.*:', '', summary(subset1)))
  q <- data.frame(subset1[0,])
  attributes(q)$row.names[1] <- "Min."
  attributes(q)$row.names[2] <- "1st Qu."
  attributes(q)$row.names[3] <- "Median"
  attributes(q)$row.names[4] <- "Mean"
  attributes(q)$row.names[5] <- "3rd Qu."
  attributes(q)$row.names[6] <- "Max."
  q$Sentiment <- p[1:6]
  q$Open <- p[7:12]
  q$Close <- p[13:18]
  q$Change_Percentage <- p[19:24]
  text1 <- paste("Summary of ", input1)
  r <- knitr::kable(q,digits=4,caption=text1)
  return (r)
}

Conclusion

The Historical data summaries and visuals shows the consistency and positive upward trend of the stock value of the mighty four. This indicates the reason why they could pass/nearly pass the One Trillion Race. Investors future sentiment and company’s future performance followed the historical market trends.
